# Media Player Tester

This is a simple html page that loads an `audio` html tag, and logs in console events triggered.
In order to work, you need to set the `URL` property n `index.js` to a valid mp3 or ogg source.

Every event log contains the type of event and a snapshot of the media element when the event was triggered.

Example:
```json
{
   "type": "event",
   "name": "suspend",
   "time": "2017-02-27T18:09:42.206Z",
   "media": {
      "currentTime": 0,
      "duration": 299.410583,
      "ended": false,
      "error": null,
      "paused": true,
      "played": [],
      "buffered": [{
         "position": 0,
         "start": 0,
         "end": 264.911
      }],
      "seekable": [{
         "position": 0,
         "start": 0,
         "end": 299.410583
      }],
      "seeking": false
   }
}
```

Once "play" happens, we receive events saying we're playing, but it's not
After ~5 seconds, we receive a "pause" and "ended" event, so it really stops
BUT a "progress" event will happen to tell us the buffer is now ready

We could then, click "play" again
